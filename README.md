# Bilibili Fitness Timer

把 B 站健身视频的片段编排成可跟练的动作、组次和休息循环。脚本使用 Tampermonkey 运行，训练计划与进度保存在浏览器本地。

## 安装

先安装 Tampermonkey，再打开下面的脚本地址并确认安装：

[安装 Bilibili Fitness Timer](https://raw.githubusercontent.com/RyanChouHua/bili-fitness-timer/main/dist/bili-fitness-timer.user.js)

脚本会在包含 BV 号的 B 站视频页自动加载。

## v2 功能

- 三区界面：训练舱显示当前动作和休息倒计时，工作台负责计划编辑，底部 Dock 提供稳定的模式入口。
- 多分组：同一个 BV 视频可维护多套训练分组，并支持新建、复制、切换、改名和删除。
- 在线时间戳库：按当前 BV 从 GitHub 时间戳库导入计划；请求失败或数据无效时不会覆盖本地内容。
- 全局进度：训练中同时显示当前动作、组次和整份计划的完成进度。
- 断点续练：刷新或重新进入匹配的 BV 视频时，可恢复未完成的训练；计划内容变化后不会错误恢复旧进度。
- 训练控制：视频片段循环、完成本组、休息倒计时、暂停/继续、跳过休息和重置训练。

## 录入语法

每行录入一个动作，基本格式为：

```text
动作名 开始时间-结束时间 组数x次数 rest休息秒数
```

示例：

```text
深蹲 00:12-00:40 3x12 rest45
俯卧撑 00:45-01:20 3x8-12 rest60
平板支撑 01:25-02:05 2组1 休息30
```

- 时间段必填，分隔符支持 `-`、`~`、`至`、`到`；时间支持 `m:ss`、`h:mm:ss` 和小数秒。
- 组数次数必填，支持 `3x12`、`4×10`、`3组8`，也可用 `3x8-12` 表示次数范围。
- 休息可省略，默认 45 秒；支持 `rest45` 或 `休息45`。
- 动作名可省略，脚本会按顺序生成名称。

## 从旧版升级

v2 沿用原仓库、安装地址和 `@updateURL`。旧用户无需卸载或重新安装，Tampermonkey 会在 `0.5.0` 发布到 `main` 后平滑升级；旧版单计划数据会自动兼容为 v2 分组计划。

升级前如需额外保险，可先在工作台导出本地计划。v2 不提供跨设备同步，训练计划和断点进度仍保存在当前浏览器中。
